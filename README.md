## Simplify-Money- Metal Prices App 

## Overview:
'''
This is a React Native (Expo) mobile application that displays live prices of precious metals like Gold, Silver, Platinum, and Palladium.

The app simulates a real-world financial product with clean UI, navigation, and proper state handling
'''
## Features:
''' 
1. Displays live metal prices.
2. Card-based UI for each metal.
3. Independent loading state per metal.
4. Error handling for API failures.
5. Navigation between Home and Details screens.
6. Shows formatted Date & Time.
7. Built using Expo Router (file-based navigation)
'''
## Tech Stack:
'''
 1. React Native 
 2. Expo Router
 3. javascript(ES6+)
 4. React Hooks(useState,useEffect)
'''
## Installation & Setup:
'''
 1. Clone the repository 
 git clone https://github.com/Venkateshgadham/Simplify-Money 
 cd frontend

 2. Install dependencies: npm install

 3. Run the app : npx expo start 
'''
 ## API :
